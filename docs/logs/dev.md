## UI/UX Pro Max Skill 安装 

cli 安装
```sh
# 1. 全局安装 CLI 工具（只需执行一次）
npm install -g uipro-cli

# 2. 进入你的项目目录
cd /path/to/your/tauri-project

# 3. 为 Opencode 安装 Skill
uipro init --ai opencode
```
